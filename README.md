# DITHERVOID // PANDORA — FIELD MANUAL

**A box that mints ghosts.** PANDORA is a self-contained generative engine that grows infinite
animated glitch loops from a single number. Nothing is drawn by hand and nothing is random twice:
every loop is fully determined by its DNA, so any loop can be resurrected byte-exactly, by anyone,
forever. The whole machine is one HTML file.

**Live box:** https://0xgreencross.github.io/PANDORA/

---

## THE MONITOR
The square screen is the loop, playing at a locked 25fps on the box's own player.
The readout under it shows the signature, seed, frame count, and file sizes.
Tap the size readout to expose playback diagnostics.

## SIGNAL — the coarse dials
- **PALETTE** — 26 fixed DITHERVOID colorways, plus **GEN∞** (the seed grows its own palette),
  **TIS∞** (palettes grown by the TIME IS SCAM rules), and **RND**.
- **SOURCE** — the body the corruption feeds on: PLASMA, RINGS, GLYPHS, SCANS, LATTICE,
  ORBIT, STRATA, TENDRIL, or AUTO (the seed chooses).
- **CORRUPTION** — the violence dial. Low is clean signal; high is the void biting back.
- **FRAMES** — loop length at 25fps.
- **ERA** — the box holds two epochs of physics. **SIGNAL** is the current engine
  (index-space corruption, dither quake, attribute clash). **CLASSIC** resurrects the
  earlier, pre-glitch aesthetic — not an imitation: the original code, proven byte-identical.
  Flick it and the same seed re-renders in the other era.

## SIGILS — 37 corruption channels
Each sigil arms one channel of destruction: tape tracking, pixel sort, attribute clash,
ransom stamps, dither quake, tape rot, and the rest of the arsenal. Lit means armed.
Combinations multiply — the same seed under different sigils is a different ghost.
**☠ OBLIVION** arms a violent random set and fires.

## TUNE — surgical control
Override what the seed decided: **MASK** (30 carving shapes × 20 styles, with size and
position), **BODIES** (up to 50 orbiting forms in 30 shapes). AUTO hands the dial back
to the seed. Everything you tune is recorded in the DNA.

## LAB — the genome room
- **SEED** — the genome number. The dice roll a fresh one.
- **DNA CARD** — the loop's complete genetic code as a short string
  (`PNDR-…`). Copy it, post it, keep it: whoever holds the card can resurrect the exact
  loop on any device, any time. The card is the artwork's proof.
- **PUBLIC VAULT** — a shared collection every visitor sees. STORE adds the current
  loop (card + thumbnail) for everyone; entries are permanent unless the curator removes them.
- **GARDENS** — your wallet's own shelf. **CONNECT WALLET** (Ethereum, one tap, no gas,
  ever) then **SAVE TO MY GARDEN**: each save is a wallet signature, publicly displayed
  under your address, curated by you alone. Plant without limit. What grows here matters
  for the next era.

## 𝕏 TRANSMISSION
The 𝕏 button opens the transmission modal: an X-safe version of the loop (full resolution
whenever it fits X's 15MB ceiling), a save button, and a **procedurally corrupted post** —
composed in a glitched alphabet, seeded so that no two transmissions in history repeat.
X cannot carry the GIF automatically: save it, compose, attach by hand, post.

## INFO MODE
Tap **?** next to the tabs, then tap anything — the box explains the control instead of
firing it. On desktop, hovering any control shows the same knowledge.
